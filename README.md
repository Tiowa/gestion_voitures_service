 Ce projet Python simule la gestion de voitures de service attribuées aux employés.

Le système permet :
- d'associer une voiture à un employé
- d'afficher les informations des employés et des voitures
- de retirer une voiture d’un employé

Règles appliquées :
- un employé ne peut avoir qu'une seule voiture
- une voiture ne peut être attribuée qu'à un seul employé
